## 2. Phase 1：ACM 冲金（2026.05 – 2026.12）

> 当前阶段，ACM 是绝对主线。退役前不考虑正式启动科研或考研复习。

### 2.1 赛季路线

```
暑期集训 ──→ 区域赛（10–11 月）──→ EC-Final（12 月）──→ 退役
```

### 2.2 ACM 训练

**目标**：区域赛金牌 → EC-Final 好成绩。

从区域银到金的跨越，通常需要：

- 队伍配合默契度显著提升（分工、读题、交题节奏）
- 中高难度题（CF 1900–2400）的稳定通过率
- 区域赛级别全套真题模拟（至少 8–10 场完整模拟）

**每周训练建议**（学期中）：

| 项目 | 频率 | 说明 |
|------|------|------|
| 个人 VP | 3–4 场/周 | CF Div1+Div2 / AtCoder ARC/AGC，赛后补题 |
| 团队模拟 | 1 场/周 | 往年区域赛真题，严格计时 5h，赛后复盘 |
| 专题刷题 | 每日 2–3 题 | 按薄弱专题滚动：图论 → 字符串 → DP → 数学 → 数据结构 |
| 队内讨论 | 每周 1 次 | 复盘模拟赛，讨论各自补题心得 |

**暑假集训（2026.7 – 2026.8）**：
- 校队集训全力参加；如无集训，自行维持每日 6–8h，保持 VP + 补题节奏
- 至少参加 2 场多校联合赛（杭电多校、牛客多校）

### 2.3 GPA：底线思维

GPA 不高已是事实，不再为保研线焦虑。但注意：

- **大二下期末别挂科**——挂科影响考研复试观感
- 保持一个「不算难看」的成绩单即可，不求排名

### 2.4 退役前「零成本信息收集」

不启动正式准备，但碎片时间可以：

- 浏览 ISCAS 官网，看看有哪些实验室、导师方向
- 知乎/考研论坛上搜「中科院软件所 考研 经验」，建立模糊认知
- 知道 408 考哪四门（DS / 计组 / OS / 计网），心里有个数

## 3. Phase 2：退役过渡期（2027.01 – 2027.06）

> EC-Final 结束后，你有 6 个月来做三件事：**定方向、找导师、打基础**。这个阶段不需要冲刺，但不能浪费。

### 3.1 确定科研方向

结合 ACM + 数媒背景，推荐方向（按契合度）：

| 方向 | 理由 | ISCAS 对应 |
|------|------|------------|
| 算法与复杂性理论 | ACM 思维无缝衔接 | 计算机科学国家重点实验室 |
| AI 系统 / 机器学习 | 热门，代码能力可迁移 | 多个实验室涉及 |
| 计算机图形学 / 可视化 | 数媒背景吻合 | 人机交互与智能信息处理实验室 |
| 软件工程 / 编译系统 | 工程能力强，代码功底扎实 | 软件工程技术中心 |

**行动**：
1. 每个方向精读 2–3 篇综述论文（2–3 周），感受哪个方向让你「愿意熬夜读下去」
2. 跟 ACM 退役学长聊——他们选了哪个方向、踩了哪些坑
3. **最迟 2027 年 3 月前做出选择**

### 3.2 联系导师

**时间窗口**：2027 年 3–5 月（大三下学期）

- 中科院导师普遍欢迎提前联系，尤其是有竞赛背景的学生
- 准备：简历（强调 ACM 成绩 + 代码能力）+ 对导师方向的初步理解
- 邮件要点：简洁、真诚、展示你读过他近两年的论文
- 多联系几位，不要只盯一个

### 3.3 科研起步

确定方向 + 联系导师后，做一件**小而完整**的事：

- 复现一篇经典论文的代码
- 在公开 benchmark 上跑实验 + 写分析
- 目标：产出一段**能写进简历、能在复试中讲清楚**的科研经历

### 3.4 考研基础复习

退役后时间充裕，考研基础可以**从容开始**：

| 科目 | 内容 | 节奏 |
|------|------|------|
| 数学一 | 高数上下册一轮 + 线代基础 | 每天 2–3h |
| 英语一 | 背单词（每天 50–100）+ 精读真题阅读 | 每天 1h |
| 408 | 不急着开始 | — |
| 政治 | 不急着开始 | — |

> 这个阶段的目标不是刷题，而是把数学和英语的底子打牢——数学一轮过完、单词背完第一遍。这对 ACM 选手来说节奏很舒服。

### 3.5 退役后学科全景

> 从 EC-Final 退役到研一开学，你需要覆盖的学科分四个层次。考研应试只是其中的一条线，真正拉开差距的是**数学功底**和**科研方向课**。

---

#### 第一层：考研必考（四科）

这些是硬门槛，分数直接决定能否进复试。

| 学科 | 覆盖内容 | 在中科院软件所考研中的地位 |
|------|----------|--------------------------|
| **数学一** | 高等数学（上下册）、线性代数、概率论与数理统计 | 满分 150，拉分主力。ACM 选手逻辑不差，但计算量需要练 |
| **英语一** | 完形、阅读、新题型、翻译、写作 | 满分 100，过线容易高分难。ACM 读题经验 = 阅读天然优势 |
| **408 计算机综合** | 数据结构、计算机组成原理、操作系统、计算机网络 | 满分 150，四门融在一张卷。ACM 选手 DS 是强项，但计组和 OS 需要从零啃 |
| **政治** | 马原、毛中特、史纲、思修、时政 | 满分 100，9 月启动即可，短期记忆科目 |

---

#### 第二层：数学功底（超越考研大纲）

ACM 冲金选手的数学不能只停在考研水平。以下学科不管是做理论还是做系统，都是底层竞争力。

| 学科 | 与 ACM/科研的关联 | 学习时机 |
|------|------------------|----------|
| **线性代数（进阶）** | 矩阵分解、特征值、SVD、PCA——机器学习全部建立在线代之上 | Phase 2（退役后立即开始） |
| **概率论与数理统计** | 考研也要考，但 ML 方向需要远超考研水平：贝叶斯推断、MLE/MAP、EM 算法、马尔可夫链 | Phase 2–3 |
| **离散数学** | 集合论、图论、组合数学、数理逻辑——ACM 选手已经有大量直觉，需要的是系统化 | Phase 2（配合算法方向） |
| **数值分析 / 计算方法** | 浮点误差、数值稳定性、迭代法——做图形学或 AI 系统都绕不开 | Phase 2–3（选学，看方向） |
| **凸优化** | 梯度下降、拉格朗日对偶、KKT 条件——ML 方向必学，理论方向加分 | Phase 3（确定 ML 方向后） |

---

#### 第三层：计算机核心（408 + 补充）

408 的四门是底线，以下列出每门的具体内容和退役后的额外补充。

| 学科 | 408 考什么 | 退役后额外该学什么 |
|------|-----------|-------------------|
| **数据结构** | 线性表、栈队列、树、图、查找、排序 | ACM 选手已远超考研要求。只需补**概念题和手写代码题**的答题规范（考研不是 OJ，要写注释和思路）。 |
| **计算机组成原理** | 数制与编码、CPU 架构、指令系统、存储器层次、总线、I/O | 额外学：**RISC-V 或 x86 汇编基础**、缓存一致性、流水线与分支预测。这些在复试面试中是高频问题。 |
| **操作系统** | 进程管理、内存管理、文件系统、I/O 管理 | 额外学：**Linux 内核基础**（进程调度、虚拟内存实现）、并发编程（锁、RCU、无锁结构）。如果你的科研涉及系统方向，这是核心。 |
| **计算机网络** | 物理层到应用层、TCP/IP、HTTP、网络安全 | 额外学：**HTTP/2 & HTTP/3 协议细节**、TLS 握手、CDN 原理。不管是做系统还是做 AI 部署，网络基础都绕不开。 |
| **编译原理**（不在 408 内） | — | 如果你选软件工程/编译系统方向，这是必修。即使不选，理解「代码→AST→IR→机器码」的管线，对任何 CS 研究生都是基本功。 |
| **数据库系统**（不在 408 内） | — | 关系代数、SQL 优化、事务与并发控制、B+ 树索引。ML 方向也需要处理数据管线。 |

#### 第四层：科研方向课（按方向选修）

确定方向后，从对应列表中选 2–3 门深入。

**算法与复杂性理论方向：**

| 学科 | 内容 | 优先级 |
|------|------|--------|
| 算法导论（CLRS 后半本） | 高级数据结构、平摊分析、网络流、NP 完全性 | ★★★ |
| 计算复杂性 | P vs NP、近似算法、随机算法、交互式证明 | ★★★ |
| 组合优化 | 线性规划、整数规划、拟阵理论 | ★★ |
| 密码学基础 | 单向函数、公钥加密、零知识证明 | ★ |

**AI 系统 / 机器学习方向：**

| 学科 | 内容 | 优先级 |
|------|------|--------|
| 机器学习（西瓜书 + 吴恩达） | 监督/无监督、决策树、SVM、神经网络、集成学习 | ★★★ |
| 深度学习 | CNN、RNN、Transformer、GAN、扩散模型 | ★★★ |
| 凸优化 | 梯度下降变体、对偶理论、收敛性分析 | ★★★ |
| 概率图模型 | 贝叶斯网络、马尔可夫随机场、变分推断 | ★★ |
| 强化学习 | MDP、Q-Learning、策略梯度、PPO | ★ |

**计算机图形学 / 可视化方向：**

| 学科 | 内容 | 优先级 |
|------|------|--------|
| 计算机图形学 | 光栅化、光线追踪、辐射度、着色模型 | ★★★ |
| 几何处理 | 网格处理、曲面重建、微分几何基础 | ★★★ |
| 数值分析 | 线性方程组求解、插值、最小二乘 | ★★★ |
| 物理模拟 | 刚体/流体模拟、有限元方法 | ★★ |
| 人机交互 | 交互设计原则、可用性评估、原型工具 | ★★ |

**软件工程 / 编译系统方向：**

| 学科 | 内容 | 优先级 |
|------|------|--------|
| 编译原理（龙书） | 词法分析、语法分析、中间代码、代码优化 | ★★★ |
| 程序分析 | 数据流分析、指针分析、抽象解释、符号执行 | ★★★ |
| 软件工程 | 需求分析、架构设计、测试方法、DevOps | ★★ |
| 形式化方法 | 模型检测、时序逻辑、定理证明 | ★ |
| 分布式系统 | 一致性协议（Paxos/Raft）、分布式事务、容错 | ★★ |

## 4. Phase 3：全力备考 + 科研并行（2027.07 – 2027.12）

> 暑假到初试，核心矛盾是「备考强度」与「科研不断档」的平衡。

### 4.1 考研复习总规划

| 科目 | 目标分数 | 主要资料 | 节奏 |
|------|---------|---------|------|
| 数学一 | 120+ | 张宇/汤家凤 + 李林 880 + 真题 + 模拟卷 | 7–8 月强化，9–10 月真题，11–12 月模拟 |
| 408 | 120+ | 王道四本 + 真题 | 7–9 月一轮，10–11 月二轮+大题，12 月真题 |
| 英语一 | 70+ | 真题 + 单词 | 全程背单词，9 月起刷真题套卷 |
| 政治 | 65+ | 徐涛/腿姐 + 肖秀荣 | 9 月启动 |

**每周时间分配参考**（暑假全职备考）：

| 时段 | 内容 |
|------|------|
| 上午 8:00–12:00 | 数学 |
| 下午 14:00–17:00 | 408 |
| 傍晚 17:00–18:00 | 英语单词+阅读 |

### 4.2 科研保持

- 和导师保持两周一次汇报
- 每周至少 8h 科研时间
- 如果在初试前产出一篇 arXiv preprint 或 workshop 论文，复试是很大加分项

### 4.3 关键节点

- **2027.09**：考研报名
- **2027.12 下旬**：初试

## 5. Phase 4：复试 + 毕设（2028.01 – 2028.06）

### 5.1 复试准备

中科院软件所复试：机试 + 面试 + 英语。

- **机试**：ACM 选手的主场优势——初试后每天刷 1–2 题保持手感即可
- **面试**：科研经历的清晰陈述（做了什么、为什么、结果如何）+ 408 基础问答
- **英语**：英文自我介绍 + 基本学术对话

### 5.2 毕设

如果导师在复试前已经接收你，毕设可以直接对齐导师方向——作为研究生阶段的预研。

## 6. 风险预案

| 风险 | 概率 | 应对 |
|------|------|------|
| ACM 冲金失败 | 中 | 不影响考研路径。用科研经历弥补复试亮点 |
| 初试未过线 | 低（ACM 选手数学和逻辑不差，且有 12 个月准备） | 调剂或二战，提前和导师沟通是否可保留名额 |
| 导师竞争激烈 | 中 | 多联系几位，不押注单一导师 |
| 科研方向选错 | 中低 | Phase 2 前两个月用于试读论文，发现不对及时换 |
